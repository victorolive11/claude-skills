# claude-skills

Collection de skills Claude Code installés dans ce repo.

## Skills installés

### superpowers (obra/superpowers)

Méthodologie complète de développement logiciel pour agents de code, construite autour d'un ensemble de skills composables et d'instructions qui garantissent que l'agent les utilise automatiquement. Inclut 14 skills couvrant TDD, debugging systématique, subagent-driven-development, git worktrees, revue de code, etc.

- Emplacement : `skills/superpowers/`
- Source : <https://github.com/obra/superpowers>
- Skills : `skills/superpowers/skills/` (14 skills)
- Commandes, agents, hooks : `commands/`, `agents/`, `hooks/` dans le dossier du plugin

Référencé depuis <https://github.com/travisvn/awesome-claude-skills>.
